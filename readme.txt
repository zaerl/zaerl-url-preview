Hyperlinks domain name preview in posts. This plugin is no longer actively developed and is saved for historical reasons.

== Description ==

zaerl URL Preview appends to all hyperlinks the domain name of the link found in the opening `<a>` tag. E.g.

[The GNU GPL](http://www.gnu.org/licenses/gpl.html "A Great License")

is translated to:

[The GNU GPL](http://www.gnu.org/licenses/gpl.html "A Great License") [gnu.org]

A custom preview template can be specified through the admin panel.

== Installation ==

Extract the zip file and just drop the contents in the my-plugins/ directory of your bbPress installation and then activate the Plugin from Plugins page.
